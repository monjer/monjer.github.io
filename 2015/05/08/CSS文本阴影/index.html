CSS文本阴影

发表于 2015-05-08

兼容性

基本上稍高版本的浏览器都支持该属性，并可以省略前缀。但IE在 IE10+ 才实现该属性。参见：CanIUse text-shadow

注意: text-shadow不会占据HTML的文档布局流。
